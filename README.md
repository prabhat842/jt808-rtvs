# JT808 RTVS

Owned JT1078 media service and browser UI for live view, control, and future playback.

This repository is intentionally separate from the JT808 device simulator and the JT808 gateway/server.

## Scope

- JT1078 media ingest
- session tracking
- browser UI for live monitoring and control
- WebRTC/fMP4 browser delivery in later phases

## Not included

- `cvnet.js`
- any unlicensed third-party RTVS bundle

## Initial layout

- `docs/` design and run notes
- `src/` service and UI implementation
